# Adhyay

AI-powered CBSE exam generator based on NCERT patterns.

## PDF font requirements (Hindi/Punjabi)

The PDF generator uses system fonts for complex scripts. Make sure these are installed on the server:

- Hindi (Devanagari): Nirmala UI or Mangal (Windows), Noto Sans Devanagari (Linux), Kohinoor Devanagari (macOS)
- Punjabi (Gurmukhi): Nirmala UI or Raavi (Windows), Noto Sans Gurmukhi (Linux), Gurmukhi MN (macOS)

If none of these fonts are available, PDFs will fall back to Helvetica and Hindi/Punjabi text may not render correctly.
